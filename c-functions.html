Key Concepts

Function Declaration

A prototype that tells the compiler about the function's name, return type, and parameters.

Function Definition

The actual implementation of the function with the function body.

Recursion

A function calling itself with a base case to prevent infinite loops.

Parameters

  • Pass by Value: Function receives a copy of the value
  • Pass by Reference: Function receives the memory address
