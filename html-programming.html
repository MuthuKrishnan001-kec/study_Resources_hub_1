HTML & Web Basics

Structure your webpages with HTML

Overview

HTML (HyperText Markup Language) is the standard language for creating web pages. This section covers elements, semantics, links, images, and forms.

Topics Covered

  • Elements & Tags: Headings, paragraphs, lists
  • Semantic HTML: header, nav, main, section, article, footer
  • Links & Images: anchor tags, img attributes
  • Tables & Lists: table, ul, ol
  • Forms: inputs, labels, buttons (see Forms page)
  • HTML5 APIs: audio, video, canvas basics
