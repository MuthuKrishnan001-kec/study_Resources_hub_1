CSS Styling

Style your pages with CSS

Overview

CSS (Cascading Style Sheets) controls the look and layout of web pages. Learn selectors, box model, layout (flexbox, grid), and responsive design.

Topics Covered

  • Selectors: element, class, id, attribute
  • Box Model: margin, border, padding, content
  • Layout: display, position, float, flexbox, grid
  • Typography: font, size, line-height
  • Responsive Design: media queries, mobile-first
  • Transitions & Animations basics

Example: Center a box with Flexbox

 .container { display:flex; align-items:center; justify-content:center; height:200px; }
.box { width:100px; height:100px; background:#0b6ab8; }
