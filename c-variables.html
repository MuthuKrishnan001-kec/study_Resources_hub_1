What is a Variable?

A variable is a named storage location that holds a value. It has a name, a type, and a value that can change during program execution.

Declaring Variables

// Syntax: datatype variableName;
int age;
float salary;
char initial;

// Declaration with initialization
int count = 10;
float temperature = 98.6;
char gender = 'M';

Variable Naming Rules

  • Must start with a letter (A-Z, a-z) or underscore (_)
  • Can contain letters, digits, and underscores
  • Cannot contain spaces or special characters
  • Cannot be a C keyword (int, float, if, etc.)
  • Case-sensitive: age and Age are different

Example Program

#include <stdio.h>

int main() {
    int age = 20;
    float gpa = 3.85;
    char grade = 'A';
    
    printf("Age: %d\n", age);
    printf("GPA: %.2f\n", gpa);
    printf("Grade: %c\n", grade);
    
    return 0;
}
