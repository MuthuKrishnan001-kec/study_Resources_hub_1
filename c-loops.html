Loops allow you to execute a block of code multiple times. C provides three types of loops: for, while, and do-while.

For Loop

The for loop is used when you know how many times you want to repeat the code.

for (initialization; condition; increment) {
    // Code to execute
}

// Example: Print numbers 1 to 5
for (int i = 1; i <= 5; i++) {
    printf("%d\n", i);
}

While Loop

The while loop continues as long as a condition is true.

while (condition) {
    // Code to execute
}

// Example: Print numbers 1 to 5
int i = 1;
while (i <= 5) {
    printf("%d\n", i);
    i++;
}

Do-While Loop

The do-while loop executes the code at least once, then checks the condition.

do {
    // Code to execute
} while (condition);

// Example
int i = 1;
do {
    printf("%d\n", i);
    i++;
} while (i <= 5);

Nested Loops

// Print a 3x3 multiplication table
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        printf("%d ", i * j);
    }
    printf("\n");
}
